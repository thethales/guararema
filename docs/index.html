    Início

    Este é um experimento com o intuito de criar um guia colaborativo sobre como fazer as coisas acontecerem na cidade. Seja encontrando um prestador de serviço ou aquela loja com aquele item super especifico. Ou ainda entendo as diversas burocracias dos serviços públicos através das experiências de quem já navegou por ali. Esta é uma biblioteca construida pela comunidade para a comunidade.
